Key Features

Adding activities to the list

The Add button is a very important in this application that allows the user to add activities to the list.

Removing activities from the list

The remove button delete any activity no more needed from the list

Activity saving

The save button when clicked saves an activity that is marked as completed to the database

About This Project

This project was inspired by the fact that people loss track of important activities because they are untracked. This projected with messaging alert system to remind the user by sending them text messages to remind them of the activity they intent to do.
